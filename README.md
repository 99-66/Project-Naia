# Project Naia(What Issue Now?)
현재 이 시간에 어떤 단어들이 가장 많이 언급되어지고 있는지 알아보기 위한 프로젝트이며
여러 채널에서 수집한 문장을 토대로 명사만을 추출하여 빈도수 상위 랭킹과 특정 단어와 연관된 단어들을 보여줍니다.
